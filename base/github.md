# github
git是版本控制软件，github是图形化管理git的网站。
# 安装
- windows下载exe双击下一步
- mac、linux一般自带
# 实操
- 创建一个github仓库
- 本地初始化项目，关联github仓库
- 本地提交，并推送
# 小结
```bash
# 将当前目录初始化为一个git仓库
git init 
# 将某个文件添加到暂存区
git add filename
# 将暂存区的内容提交到本地仓库
git commit -m "提交的注释"
# 将本地仓库推送到远程（github）仓库
git push origin master
```
